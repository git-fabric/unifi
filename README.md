# @git-fabric/unifi
UniFi fabric app — hosts, sites, and devices via the UI.com Cloud API as a composable MCP layer. Part of [git-fabric](https://github.com/git-fabric).
9 tools: unifi_health, unifi_list_hosts, unifi_get_host, unifi_list_sites, unifi_get_site, unifi_list_devices, unifi_get_device, unifi_network_status, unifi_debug
Env: UNIFI_API_KEY (required), UNIFI_API_BASE (default https://api.ui.com), UNIFI_API_VERSION (default v1)
